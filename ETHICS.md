# Ethics and Privacy Impact Assessment (PIA)

Даний документ описує етичні аспекти та аналіз впливу на приватність для системи моніторингу Smart City Waste Management.

## 1. Privacy Impact Assessment для геолокаційних даних
* **Об'єкт моніторингу**: Система відстежує виключно стаціонарні об'єкти комунальної власності — сміттєві контейнери.
* **Ризики**: Існує мінімальний ризик аналізу патернів життєдіяльності громадян на основі частоти заповнення баків у конкретних локаціях.
* **Заходи захисту**: Використання анонімізованих координат без прив'язки до персональних даних користувачів та захист каналів передачі даних через TLS.

## 2. GDPR Compliance
* Процеси збору та обробки даних не стосуються ідентифікації фізичних осіб (імена, фото, контактні дані не збираються).
* Доступ до системи управління та маршрутів обмежений відповідно до ролей (Admin, Driver) за допомогою JWT аутентифікації.

## 3. Політика відкритих даних
* Система підтримує надання агрегованих та анонімізованих даних для екологічних досліджень, що сприяє розвитку міської інфраструктури без загрози для приватності громадян.
