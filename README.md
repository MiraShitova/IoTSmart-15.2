# Smart Greenhouse IoT System

Цей проєкт реалізує автоматизовану систему моніторингу теплиці на базі OpenHAB.

## Архітектура
* **OpenHAB**: Центр керування та логіки (DSL Rules).
* **InfluxDB v2**: База даних часових рядів для зберігання показників (Persistence).
* **Grafana**: Платформа для візуалізації та аналізу даних.
* **Docker**: Контейнеризація всіх компонентів системи.



## Функції
* **Автоматична симуляція**: Генерація значень температури та вологості через DSL Rules.
* **Збереження даних**: Запис показників у бакет `15.1` організації `my_iot_org`.
* **Моніторинг**: Відстеження `Virtual_Temp` та `Greenhouse_Hum` у реальному часі.

## Швидкий старт
1. **Налаштуйте токен**: Вкажіть діючий токен у файлі `influxdb.cfg`: `token=Ваш_Токен`.
2. **Запустіть стек**: Виконайте команду `docker-compose up -d` у терміналі.
3. **Панелі керування**:
   * OpenHAB: `http://localhost:8080`
   * InfluxDB: `http://localhost:8086`
   * Grafana: `http://localhost:3000`

## Troubleshooting
* **Перевірка логів**: Для перегляду роботи OpenHAB використовуйте `docker-compose logs -f openhab`.
* **Активні симуляції**: Щоб знайти дані в логах, введіть `Select-String "Simulated Temp" в openhab.log`.
* **Docker Network**: Переконайтеся, що в `influxdb.cfg` вказано `url=http://influxdb:8086` для зв'язку між контейнерами.
